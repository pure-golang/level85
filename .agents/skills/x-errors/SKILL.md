# Ошибки

Этот skill больше не владеет общими правилами error handling. Для зрелого Go это базовая практика.

Здесь остаются только локальные дельты:
- error messages на английском и со строчной буквы
- для строковых значений предпочитай `%q`
- exported sentinel errors по умолчанию нежелательны; предпочитай внутренний sentinel, типизированную ошибку или `errors.Is/As`

Если задача требует просто обернуть или проверить ошибку, отдельный skill обычно не нужен.
