# Дефис или подчёркивание

Этот skill больше не считается owner-skill.

Короткое правило:
- Go-пакеты и импортируемые каталоги — без `-`
- `.go` файлы — через `_`
- внешние артефакты, markdown, yaml, scripts — через `-`
